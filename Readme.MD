# 🛍️ Retail Nexus – AI-Powered Retail Management System

Retail Nexus is an AI-powered retail management system developed as a college project. It helps retailers efficiently manage products, customers, sales, and business operations through a modern, user-friendly web interface.

## 🌐 Live Demo
https://retailnexussai.netlify.app/

## 💻 GitHub Repository
https://github.com/varshininerella08-sweety/RETAIL-NEXUS

## 🚀 Features

- Product Management
- Customer Management
- Sales Tracking
- AI-Powered Retail Dashboard
- Responsive Design
- User Authentication
- Contact Form
- Modern UI/UX

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Netlify (Hosting)

## 🎯 Purpose

The objective of Retail Nexus is to demonstrate how AI-powered retail management can simplify inventory handling, customer management, sales tracking, and decision-making for retail businesses.
